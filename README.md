# Fastype

Fastype - A Fast Modern Text Editor

## What's Fastype?

Fastype is a fast modern text editor.

## Manual Build

* Linux/UNIX

```
$ git clone --recurse-submodules https://github.com/fastype/fastype.git
$ cd fastype
$ ./build
```

* Windows

```
$ git clone --recurse-submodules https://github.com/fastype/fastype.git
$ cd fastype
$ build.bat
```
